Projects

Capstone - Osmosis

My capstone project for my software engineering course is a messaging service that leverages the React framework and Material-UI (MUI) on the client side. On the server side, it utilizes Node.js with socket.io and Express for fast and secure messaging. This project significantly enhanced my proficiency in JavaScript, emphasizing its application in creating a responsive and dynamic user interface. Additionally, the project involved database management for storing user information, groups, and messages, further solidifying my skills in database creation and administration.

Book Database

The Book Database Project showcases my proficiency in Node.js, MongoDB, and Express. Utilizing Node.js for the server, the project fetches and stores book information from an external API into a MongoDB database. Notably, users can easily create and delete books directly from the database, streamlining data management. This project has deepened my understanding of backend development, API integration, and database operations.

My Lists

The "My List Project" offers an intuitive platform for organizing individuals with a focus on simplicity and flexibility. Through this project, I honed my skills in React, Material-UI (MUI), and Vite, gaining proficiency in crafting responsive user interfaces and optimizing performance. Learning JSX, the coding language inherent to React, was pivotal in creating a seamless and user-friendly experience. The functionalities of the website include effortless card creation and editing, allowing users to customize information from basic details to personalized notes and viewing other popular lists on a separate page.

National Park API

In my National Park project, I seamlessly integrated with an external API to retrieve and process information, presenting it on dynamic and responsive cards while also implementing sorting functionality for these cards based on categories. This project also served as a valuable opportunity to expand my knowledge of CSS, as I crafted an extensive style sheet to enhance the project's visual appeal. Additionally, I developed a form of data visualization by incorporating Chart.js on a separate page to dynamically showcase API information through graphs, providing me with practical experience in leveraging charting libraries for informative and visually appealing displays.

Random Dice Generator

My initial major project in the software engineering course was a random dice generator, emphasizing fundamental HTML concepts with a basic implementation of JavaScript. This project provided a hands-on introduction to web development, honing my skills in crafting structured and responsive user interfaces.
